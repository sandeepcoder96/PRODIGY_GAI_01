# Text Generation using GPT-2

This project demonstrates text generation using the GPT-2 transformer model.

## Tools Used
Python
HuggingFace Transformers

## Description
The GPT-2 model is used to generate coherent and contextually relevant text based on a given prompt.

The model was loaded using the Hugging Face Transformers library and used to generate human-like text output.

## Output
The GPT-2 model successfully generated a continuation of the prompt.
